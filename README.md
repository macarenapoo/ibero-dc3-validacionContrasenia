# Validación de Contraseña

### Objetivo
Validar una contraseña para asegurarse que tenga:
- Al menos 8 caracteres
- Al menos 1 letra mayúscula
- Al menos 1 letra minúscula
- Al menos 1 número
- Al menos 1 caracter especial
